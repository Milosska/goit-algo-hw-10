# Інструкція щодо використання репозиторію

1. Склонуйте репозиторій

```bash
git clone [repository-url]
cd goit-algo-hw-10
```

2. Перейдіть у директорію проєкту та активуйте віртуальне середовище

```bash
 python -m venv .venv
 # On Windows:
 .venv\Scripts\activate
 # On Unix/MacOS:
 source .venv/bin/activate
```

3. Встановіть необхідні залежності

```bash
# Install all dependencies (including development tools)
pip install -r requirements.txt
```

4. Запустіть головний файл main.py

```bash
# Install all dependencies (including development tools)
python task_1/main.py
```

# Завдання 1. Оптимізація виробництва

У завданні розв’язано задачу лінійного програмування з використанням бібліотеки PuLP.

## Опис завдання:

Компанія виробляє два види напоїв: "Лимонад" і "Фруктовий сік". Для виробництва цих напоїв використовуються різні інгредієнти та обмежена кількість обладнання. Задача полягає у максимізації виробництва, враховуючи обмежені ресурси.

#### Обмеження ресурсів:

- Вода — 100 од.
- Цукор — 50 од.
- Лимонний сік — 30 од.
- Фруктове пюре — 40 од.

#### Вимоги до виробництва продуктів:

- Лимонад: вимагає 2 од. "Води", 1 од. "Цукру" та 1 од. "Лимонного соку".
- Фруктовий сік: вимагає 2 од. "Фруктового пюре" та 1 од. "Води".

## Підхід до розв’язання

- Задача змодельована як лінійна оптимізаційна модель.
- Використано цілочисельні змінні для кількості кожного напою.
- Цільова функція — максимізація суми вироблених напоїв.
- Усі ресурсні обмеження задані у вигляді лінійних нерівностей.
- Для розв’язання застосовано стандартну Python бібліотеку PuLP.

## Висновки

Лінійне програмування є ефективним інструментом для задач оптимізації ресурсів. Використання PuLP дозволяє компактно описати модель та швидко отримати оптимальне рішення навіть при наявності кількох обмежень.

# Завдання 2. Реалізація алгоритму пошуку визначеного інтеграла за допомогою методу Монте-Карло.

Програмно реалізовано алгоритм пошуку визначеного інтеграла за допомогою методу Монте-Карло. Код виконується, повертає значення інтеграла та візуалізує алгоритм пошуку для наочної демонстрації принципу роботи алгоритму.

## Опис завдання:

Обчисліть значення інтеграла функції за допомогою методу Монте-Карло.Перевірте правильність розрахунків, щоб підтвердити точність методу Монте-Карло, шляхом порівняння отриманого результату та аналітичних розрахунків або результату виконання функції quad.

## Підхід до розв’язання

- Згенеровано випадкові точки в прямокутнику:

  - по осі x: від 0 до 2
  - по осі y: від 0 до f(2)=4

- Для кожної точки перевіряється, чи лежить вона під кривою
- Значення інтеграла обчислюється як:
  **площа прямокутника × (кількість точок під кривою / загальна кількість точок)**
- Аналітичне значення інтеграла обчислено за допомогою функції `quad` з підмодуля `scipy.integrate`
- Створено графік з відображенням випадкових точок, згенерованих методом Монте-Карло

## Висновки

Метод Монте-Карло дає наближений результат, точність якого зростає зі збільшенням кількості випадкових точок. Отримане значення інтеграла є близьким до аналітичного результату, повернутого за допомогою scipy.integrate.quad. Метод простий у реалізації та універсальний, але менш точний і повільніший за аналітичні методи для простих функцій.
